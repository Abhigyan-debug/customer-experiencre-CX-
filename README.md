## 🌟 Overview

The **Customer Experience (CX) Analytics Platform** is a full-stack AI-powered web application that analyzes customer feedback and delivers real-time, data-driven insights.

It leverages **Machine Learning + NLP** to help businesses understand user sentiment, identify pain points, and improve decision-making.

---

## 🎯 Problem Statement

Organizations receive large volumes of customer feedback but struggle to:

* Extract meaningful insights
* Understand sentiment at scale
* Act quickly on customer issues

👉 This platform solves that using **automation + AI intelligence**.

---

## 💡 Solution

Our system:

* Processes customer feedback in real-time
* Applies ML-based sentiment classification
* Stores and visualizes insights
* Provides actionable analytics via dashboard

---

## 🧠 Key Features

✨ **AI Sentiment Analysis**

* Classifies feedback into:

  * ✅ Positive
  * ❌ Negative
  * ⚪ Neutral

📊 **Interactive Dashboard**

* Displays trends and analytics
* Easy-to-understand UI

⚡ **Real-Time Processing**

* Instant prediction and storage

🔐 **User Authentication**

* Secure login & session handling

📁 **Data Persistence**

* Feedback stored for historical analysis

## 🚀 Installation & Setup

### 🔧 Prerequisites

* Python 3.x
* pip

### ⚙️ Steps

```bash
# Clone repository
git clone https://github.com/your-username/cx-analytics.git

# Move into project
cd cx-analytics

# Create virtual environment (recommended)
python -m venv venv

# Activate (Windows)
venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt

# Train ML model
python model.py

# Run application
python app.py
```
